# Dependencies
- Neovim 0.5+
- lazygit
- ripgrep
- etags (Optional)

# Installation

## Ubuntu

```sh
sudo add-apt-repository ppa:neovim-ppa/unstable
sudo apt-get update
sudo apt-get install neovim

# If you already have a nvim config, please backup first.

git clone git@github.com:otavioschwanck/neovim-on-reails.git ~/.config/nvim
```
